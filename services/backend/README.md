4. SQLite
5. Swagger

## Start dev server

1. Install Bun for you OS from [here](https://bun.sh/)

2. Install project dependencies
```bash
bun install
```

3. Start the dev server
```bash
bun run dev
```

## Building

To start a production version of your app:

```bash
bun ./src/index.ts
```
