# Frontend

1. Bun
2. SvelteKit
3. Tailwind
4. Shadcn-svelte

## Start dev server

1. Install Bun for you OS from [here](https://bun.sh/)

2. Install dependencies
```bash
bun install
```

3. Start the dev server
```bash
bun run dev
```

## Building

To create a production version of your app:

```bash
bun run build
```

You can preview the production build with `
``` bash
bun run preview
```
